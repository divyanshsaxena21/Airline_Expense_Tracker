# Airline Expense Tracker

live link - https://flask-course-ml-project-peti.onrender.com

## About

This is a ML model which takes some user input like:
- Airline
- Date of Journey
- Source
- Destination
- Departure Time
- Arrival Time
- Duration
- Number of Stops
- Additional Info
